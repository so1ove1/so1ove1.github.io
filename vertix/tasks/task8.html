Об алгоритме нахождения кратчайших путей

Алгоритм Дейкстры — это алгоритм поиска кратчайшего пути от одной вершины графа до всех остальных. Алгоритм работает только для графов с неотрицательными весами рёбер.

Основные шаги алгоритма:

  1. Инициализация: расстояние до начальной вершины = 0, до всех остальных = бесконечность
  2. Отметить все вершины как непосещённые. Создать множество непосещённых вершин
  3. Для текущей вершины рассмотреть все соседние вершины
  4. Для каждого соседа вычислить новое расстояние через текущую вершину
  5. Если новое расстояние меньше, чем ранее сохранённое, обновить его
  6. Отметить текущую вершину как посещённую
  7. Выбрать непосещённую вершину с наименьшим расстоянием и повторить с шага 3

Применения алгоритма:

  • GPS-навигация и системы маршрутизации
  • Маршрутизация в компьютерных сетях
  • Транспортная логистика
  • Решение задач оптимизации в различных областях
