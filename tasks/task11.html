Задача 11: Определите, является ли система булевых функций полной. Если система неполна, укажите замкнутые классы, которым она принадлежит.

Полнота системы функций

Проанализируйте предложенный набор функций и определите его полноту.

Информация

Система булевых функций называется полной, если с помощью суперпозиции этих функций можно получить любую булеву функцию.

Замкнутые классы:

  • T0 - функции, сохраняющие 0
  • T1 - функции, сохраняющие 1
  • S - самодвойственные функции
  • M - монотонные функции
  • L - линейные функции
Верно: 0
Всего: 0
