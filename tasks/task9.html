Построение СКНФ

Задача 9: Пользователь вводит вектор функции. Система строит СКНФ.

Построение СКНФ

Введите вектор булевой функции для построения её совершенной конъюнктивной нормальной формы.

Информация

СКНФ (совершенная конъюнктивная нормальная форма) — это представление булевой функции в виде конъюнкции дизъюнкций, где в каждую дизъюнкцию входят все переменные данного набора.

Введите корректный вектор (только 0 и 1)
