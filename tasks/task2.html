Получение остаточной функции

Задача 2: На вход — вектор функции, значение аргумента (0 или 1) и номер аргумента, на выход — соответствующая остаточная функция

Получение остаточной функции

Введите вектор булевой функции, значение аргумента (0 или 1) и номер этого аргумента для получения соответствующей остаточной функции.

Информация

Остаточная функция получается из исходной функции путем фиксации значения одного из аргументов.

Введите корректный вектор (только 0 и 1).
Введите корректный номер аргумента
