Построение ДНФ

Задача 12: Реализация одного из алгоритмов построения ДНФ.

Алгоритм Квайна-МакКласки

Введите вектор булевой функции для построения ДНФ с помощью алгоритма Квайна-МакКласки.

Информация

Алгоритм Квайна-МакКласки используется для минимизации булевых функций. Он находит все простые импликанты функции, а затем выбирает минимальное подмножество, покрывающее все единицы функции.

Введите корректный вектор (только 0 и 1)
